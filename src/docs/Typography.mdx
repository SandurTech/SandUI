# Typography

SandUI uses **Inter** as its primary typeface and **Outfit** for display headings.

## Font Stack

```scss
$font-main:
  "Inter",
  system-ui,
  -apple-system,
  sans-serif;
$font-display: "Outfit", "Inter", system-ui, sans-serif;
```
