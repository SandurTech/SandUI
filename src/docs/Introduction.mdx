## Vision

SandUI exists so that every product under SandurTech feels like it belongs to the same family.
A user moving between tools should never feel a visual disconnect. This library enforces that through:

- **Strict design tokens** — colors, spacing, and typography locked to a single source of truth
- **Atomic architecture** — components organized from Atoms to Templates
- **Theme-aware rendering** — light and dark modes via CSS custom properties + React context
- **Extensible by default** — every component accepts `className` for overrides

## Getting Started

### Installation

```bash
npm install @sandurtech/sandui
```

### Usage

```jsx
import { SandButton, SandCard, SandThemeProvider } from "@sandurtech/sandui";

function App() {
  return (
    <SandThemeProvider>
      <SandCard>
        <h2>Welcome</h2>
        <SandButton variant="primary">Get Started</SandButton>
      </SandCard>
    </SandThemeProvider>
  );
}
```
