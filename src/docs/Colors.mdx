# Color System

SandUI's color system is built around three brand colors from the SandurTech ecosystem,
designed for both light and dark themes.

## Why These Colors?

The palette is deliberately minimal: **three colors** that work across all SandurTech products.
The accent orange provides energy, the charcoal primary ensures readability, and the silver
secondary gives breathing room.
